# Dental Problem Detection System

This is a simple Flask web application that helps users identify possible dental problems based on selected symptoms.

## Features

* Admin login system
* Admin can add dental problems
* Admin can update and delete dental problems
* Users can select symptoms
* System suggests possible dental problems
* View detailed information about each problem

## Technologies Used

* Python
* Flask
* HTML
* CSS
* MySQL

## Project Structure

* main.py – Flask backend
* HTML templates for user interface
* MySQL database for storing dental problems

## How to Run the Project

1. Install Python
2. Install Flask
3. Setup MySQL database
4. Run the project using:

python main.py

Then open:
http://127.0.0.1:5000/
