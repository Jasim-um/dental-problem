About Dental Problem Detection System

This system helps users detect possible dental problems based on selected symptoms. Users can choose symptoms such as tooth pain, bleeding gums, sensitivity, and bad breath.

The system analyzes the selected symptoms and suggests possible dental conditions from the database. Users can also explore detailed information about dental problems including symptoms, causes, treatment, prevention, and severity.

An admin panel allows administrators to manage the dental problem database by adding, editing, and deleting records.
